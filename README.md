# SpringBootJava v2.1.3

Starting point for Spring Boot Java and Docker

### Run tests with info

gradle clean test --info

### Start SpringBoot

gradle build && java -jar build/libs/SpringBootJava.jar

### Start with Docker

bash docker-start.sh

### URL

http://localhost:8080/
