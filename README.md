# SpringBootJava v2.1.3

Example microservice with Spring Boot, Java and Docker

### Run tests with info

gradle clean test --info

### Update Gradle

gradle wrapper --gradle-version 7.3.3 --distribution-type al

### Run

./run.sh

### Run Docker

./docker-run.sh

### URL

http://localhost:8080/
