# SpringBootJava v2.1.3.RELEASE

Starting point for Spring Boot Java and Docker

### Run tests with info

./gradlew clean test --info

### Start SpringBoot

./gradlew build && java -jar build/libs/SpringBootJava.jar

### Start with Docker

bash docker-start.sh

### URL

http://localhost:8080/
